# CareerMatch - Acceptance Tests

Este repositorio contiene la aproximación a pruebas de aceptación del proyecto CareerMatch, desarrolladas en formato Gherkin para el cuarto avance del trabajo final del curso Ingeniería de Requisitos.

## Proyecto

CareerMatch es una plataforma orientada a estudiantes universitarios de últimos ciclos y recién egresados que buscan prácticas, primer empleo o posiciones junior. La solución propone recomendaciones laborales personalizadas, compatibilidad perfil-vacante, detección de brechas de habilidades y seguimiento de oportunidades.

## Contenido del repositorio

Los archivos `.feature` se encuentran en la carpeta `/features/`.
